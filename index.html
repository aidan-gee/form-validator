Demo Form

The start method below is run on window load, this runs validateForm on submit.

validator.start('demoForm',
 ['name','email','uk-postcode','country'], 
function(form){ //callback });

Demo Form 2

The validateForm can be run manually, this runs the validation and returns a form object.

validator.validateForm('demoForm', 
['name','email','uk-postcode','country']);

Demo Input

The validateInput method validates a single input by id. This will use the inputs name (like the above examples) to apply different validations. Note: If there is no name then just the default blank validation will run.

validator.validateInput('demoInput');
